# Well-Scenario グラフ表示・人手アノテーション機能の実装

## タスクの概要
シナリオのメトリクススコアをグラフ表示し、グラフ上で点を手動調整できる機能を追加する。手動調整したスコアは機械によるアノテーションとは別にJSON保存する。
